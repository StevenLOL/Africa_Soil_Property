2. Instruction
2.1 Prequiste packages
- All the scripts run on Ubuntu 12.04 OS
- packages need to be installed:
	+) python2.7 (installed by default on ubuntu 12.04)
	+) python-py (could be installed by default)
		$sudo apt-get install python-pip
	+) build-essential
		$sudo apt-get install build-essential
	+) pandas nose
		$sudo pip install pandas nose
	+) numpy, scipy
		sudo pip install scipy numpy
	+) sklearn
		$sudo pip install scikit-learn
	+) r
		$sudo apt-get install r-base
